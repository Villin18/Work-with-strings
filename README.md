# Редактор строк

Простая консольная утилита для работы с текстом.

## Функциональность

- **Шифрование** - шифр Цезаря со сдвигом
- **Дешифрование** - расшифровка шифра Цезаря  
- **Анализ текста** - подсчет длины и пробелов
- **Форматирование** - изменение регистра текста

## Установка

```bash
python main.py
